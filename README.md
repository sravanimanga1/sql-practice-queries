# 🗄️ SQL Practice Queries

## 📖 Description
This project contains basic SQL queries for practicing database concepts such as SELECT, WHERE, ORDER BY, GROUP BY, and JOIN.

## ✨ Features
- Basic SQL queries
- Filtering data
- Sorting records
- Using aggregate functions
- Beginner-friendly examples

## 💻 Technologies Used
- MySQL
- SQL

## 🚀 How to Use
1. Open MySQL Workbench or any SQL editor.
2. Create a sample database.
3. Run the queries one by one.
4. Observe the output.
